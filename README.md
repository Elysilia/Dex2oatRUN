>这是一个magisk模块，它调用了系统中对安卓系统下应用的dex文件进行编译优化的程序。通过合理的dex2oat的编译优化，可以大大提高应用启动的速度和使用手机过程的的流畅度。

## 解释

模块根据配置在**系统更新**及**安装应用**后*定时或开机*执行dex2oat（第一次安装完模块也会执行）。

## 测试环境

`MIUI 14`
`Android 12`

## 配置

- 配置及日志相关文件都在此文件夹下：
  
  ```
  
  /data/adb/Dex2oatRUN
  
  ```
